# Advanced CSV Viewer

A modern and powerful CSV file viewer and editor built with React and TypeScript.

## Technologies

- **[Vite](https://vitejs.dev/)** - Lightning-fast build tool
- **[React 18](https://react.dev/)** - UI library
- **[TypeScript](https://www.typescriptlang.org/)** - Static typing
- **[Material UI](https://mui.com/)** - UI components
- **[AG Grid](https://www.ag-grid.com/)** - Advanced data grid
- **[Redux Toolkit](https://redux-toolkit.js.org/)** - State management
- **[React Query](https://tanstack.com/query)** - Async data management

## Installation

```bash
# Clone the repository
git clone https://github.com/your-username/advanced-csv-viewer.git

# Enter the directory
cd advanced-csv-viewer

# Install dependencies
npm install
```

## Usage

```bash
# Development
npm run dev

# Production build
npm run build

# Preview production build
npm run preview

# Lint
npm run lint
```

Access `http://localhost:5173` in your browser.
